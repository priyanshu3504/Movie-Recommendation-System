# 🎬 Movie Recommender System

A Machine Learning-based Movie Recommender System built using **Python** and deployed using **Streamlit**.
This application suggests movies based on similarity, helping users discover new content easily.

---

## 🚀 Live Demo

👉 Click https://movie-recommendation-system-tryme.onrender.com to try the app

---

## 📌 Features

* 🔍 Select a movie and get top recommended movies
* 🎥 Displays movie posters using TMDB API *(feature temporarily facing issues due to API/response delays)*
* ⚡ Fast recommendations using precomputed similarity
* 🌐 Simple and interactive UI built with Streamlit

## 🧠 How It Works

* Movie dataset is processed and cleaned.
* Features like genres, keywords, cast, and crew are combined.
* Text vectorization is applied to convert data into numerical form.
* Cosine similarity is used to calculate similarity between movies.
* Top similar movies are recommended based on user selection.
